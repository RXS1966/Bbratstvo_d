# AI Avatar Exam

Проект AI-экзаменатора компетенций менеджеров по продажам.

## Запуск проекта

Перейти в папку проекта:

cd Avatar

Активировать виртуальное окружение:

..venv\Scripts\Activate.ps1

Установить зависимости:

pip install -r requirements.txt

В файле `.env` указать свой OpenAI API ключ:

OPENAI_API_KEY=ваш_ключ_openai

Запустить сервер:

uvicorn app.main:app --reload

После запуска открыть в браузере:

http://127.0.0.1:8000

Отчет и диаграммы пока как показ загружаются по кнопке Показать результаты-итоги, Диаграммы- пример будующих диаграмм.
