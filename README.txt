Variant_3 — хронология действий в этой папке
============================================

Назначение варианта (из переписки): monorepo с общим пакетом UI (packages/ui)
и приложением на Next.js или Vite — единые компоненты и токены при росте
числа клиентов.

Рядом лежит README.md — тот же журнал в Markdown.
